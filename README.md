SHOPEZ - E-Commerce Web Application

Project Overview

SHOPEZ is a full-stack e-commerce web application developed using React, Node.js, Express.js, and MongoDB.

Features

- User Registration & Login
- Product Listing
- Product Details Page
- Search Products
- Category Filtering
- Add to Cart
- Checkout Page
- Payment Page
- Order Confirmation
- User Profile
- Contact Page
- Responsive UI
- MongoDB Database Integration

Technology Stack

Frontend

- React.js
- Vite
- React Router DOM
- Axios

Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
